# Terminology
- Instance: a running application, which is an aggregation of templates, managed by a configuration
- Template: a template contains files that are used in instances
- Configuration: a configuration is a set of parameters are used to generate instances
- InstanceGroup: an instance group is a collection of instances, which are generated from the same configuration
- TemplateGroup: a template group is a collection of templates, which can be used to take a template at random, or to group templates together for a specific purpose

- Extensions: extensions are used to extend the functionality of the application, they can be used to add new features, or to modify existing features
